# Better ZetaMac

A local offline replica of the [Arithmetic Game by ZetaMac](https://arithmetic.zetamac.com/), with an added score tracking chart. All credit for the original game concept, design, and mechanics goes to ZetaMac.

---

## How to launch

1. Download or clone this repository
2. Open the `zetamac` folder
3. Double-click `index.html`

No internet connection required. No installation needed.

---

## Features

### Game
- Solve as many arithmetic problems as you can before time runs out
- Choose which operation types to include: **Addition**, **Subtraction**, **Multiplication**, **Division**
- Customize number ranges for addition and multiplication
- Set duration: 30s / 60s / 120s / 300s / 600s (default: 120s)

### Controls
- **Cmd+R** (Mac) or **Ctrl+R** (Windows/Linux) — restart the current round instantly with the same settings, without going back to the settings page
- Answers are accepted as you type — no need to press Enter

### Score history
- A line chart below the settings tracks your **daily average score** over time
- Only completed games (where the timer runs to zero) are counted
- Data is saved in your browser's local storage — it persists across sessions
- **Clear history** button removes all saved data
